Hermes Java SDK Architecture

OpenAI-compatible Java client with Skill activation, retry, and thread-safety

SDK Features

  • • Builder 模式 immutable 配置
  • • 4 级异常体系（Network/Auth/Api/Base）
  • • 指数退避重试（1s/2s/4s）
  • • CopyOnWriteArrayList 线程安全
  • • CompletableFuture 异步支持
  • • HTTPS 强制 + Token 脱敏

Skill Activation

  • • activateSkill(name, task) 一行调用
  • • System Prompt 激活方案（可靠）
  • • skill_view() 动态加载 SKILL.md
  • • Deterministic 触发（无 AI 模糊匹配）
  • • 支持 karpathy-principles 等自定义 Skill
  • • 异步 executeAsync() 不阻塞

Integration

  • • Spring Boot 2.x 自动配置
  • • OpenAI 兼容 API（/v1/completions）
  • • REST Controller 直接暴露
  • • Maven Javadoc + Surefire 测试
  • • 健康检查 healthCheck()
  • • HermsProperties @ConfigurationProperties
